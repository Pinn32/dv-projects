1.1 Dropping Missing Values

Use dropna() to remove rows with missing values in the columns required for analysis.

# drop rows where 'text' or 'retweet_count' is missing (value is NA or NaN)
df_clean = df.dropna(subset=["text", "retweet_count"])

Dropping Missing Values:

  • Use df.dropna() to remove rows containing missing values (NA or NaN).
  • By default, it removes any row with at least one missing value.
  • Specify subset as a list of column names to only consider missing values in those columns for removal.
  • Use parentheses () or square brackets [] for the subset list.

Note: For data preprocessing, selectively drop rows with missing values only in columns relevant to your analysis to maintain data accuracy.

df.shape # view original shape (num of rows, num of cols)
(430, 14)
df_clean.shape # shape after dropping rows with missing values
(429, 14)

1.2 Filling Missing Values

Use fillna() when a missing value should be replaced rather than removed.

# fill missing values in 'mentions' with a placeholder string
df["mentions"] = df["mentions"].fillna("No Mentioned User")

Filling Missing Values:

  • Use df.fillna(value) to replace missing values in a specific column with a placeholder, such as a string.
  • Assign the result back to the column to update the DataFrame.

Use replace() to convert placeholder values into a more useful representation.

# replace the placeholder string with 0

  • Use df.replace(old_value, new_value) to substitute specific values in a column.
  • This is useful for converting placeholders into more appropriate representations, like numbers.

6 Grouping and Aggregation

Use groupby() with agg() to compute summary statistics by group. The example below calculates total retweets and average follower count for each user.

# group by 'user', then aggregate 'retweet_count' with sum 
# and 'user.followers_count' with mean
df_agg = df.groupby("user").agg(
    {"retweet_count": "sum", "user.followers_count": "mean",}
)

Grouping and Aggregation:

  • Use df.groupby('column').agg({'col1': 'sum', 'col2': 'mean'}) to group by a column and compute aggregate statistics.
  • Verify the number of groups using df[~df.duplicated(subset='column')].shape or df['column'].nunique().

7 Melting and Pivoting

7.1 Melting Data

Melting: wide → long

Use melt() to convert wide data into long format for analysis and plotting.

# create a sample wide-format DataFrame with pet types and counts
df_wide = pd.DataFrame(
    data=[
        ["cat", 4, 5],
        ["dog", 7, 4],
        ["rabbit", 2, 3],
    ],
    columns=["pet", "male", "female"],
)

df_wide # view it
pet male female
0 cat 4 5
1 dog 7 4
2 rabbit 2 3

Without arguments, melt() collapses all columns into variable-value pairs.

df_wide.melt() # melt all cols into variable-value pairs (pivot longer)
variable value
0 pet cat
1 pet dog
2 pet rabbit
3 male 4
4 male 7
5 male 2
6 female 5
7 female 4
8 female 3

Specify identifier and value columns for a structured long-format result.

melted_df = pd.melt(
    df_wide,
    id_vars=["pet"], # keep 'pet' as identifier column
    value_vars=["male", "female"], # value columns
    var_name="sex", # name for the new variable column
    value_name="count", # name for the new value column
)

7.2 Pivoting Data

Pivot: long → wide

Pivot the long data back to wide format:

df_pivot = melted_df.pivot(
    index="pet", # identifier column becomes index
    columns="sex", # categories in variable column becomes new columns
    values="count", # value column becomes new values
).reset_index() # reset index to turn 'pet' back into a column

df_pivot.columns.name = None # remove the name of the columns index

df_pivot # view the pivoted DataFrame
pet female male
0 cat 5 4
1 dog 4 7
2 rabbit 3 2

Use pivot_table() instead of pivot() when there are duplicate index/column combinations, and specify an aggregation function like aggfunc='mean' to summarize duplicates.

For example, if we have duplicate pet entries:

# create a sample DataFrame with duplicate pet entries
df_long = pd.DataFrame(
    data=[
        ["cat", "male", 4],
        ["cat", "female", 5],
        ["dog", "female", 7],
        ["dog", "male", 4],
        ["rabbit", "male", 2],
        ["rabbit", "female", 3],
    ],
    columns=["pet", "sex", "count"],
)
df_long # view the long-format DataFrame with duplicates
pet sex count
0 cat male 4
1 cat female 5
2 dog female 7
3 dog male 4
4 rabbit male 2
5 rabbit female 3

To pivot this long data back to wide format, we can use pivot_table() to handle duplicates:

# create a pivot table that averages counts for duplicate entries
(df_long
.pivot_table(
    index="pet",
    columns="sex",
    values="count",
    aggfunc="mean", # average counts for duplicate entries
)
.rename_axis(None, axis=1) # remove the name of the columns headers
.rename_axis(None) # remove the name of the index
)
female male

dog 7.0 4.0

- Specify `subset` as a list of columns to check for duplicates based on those columns only.  
- Use `keep=False` to mark all duplicates.  

Find users who posted multiple times on the same date by specifying a subset of columns.

```{python}
# find duplicated rows based on 'user' and 'date' columns (keep all occurrences)
df[df.duplicated(subset=["user", "date"], keep=False)].head(2)
```

# Melting and Pivoting

## Melting Data

**Melting: wide → long**  

Use `melt()` to convert wide data into long format for analysis and plotting.

```{python}
# create a sample wide-format DataFrame with pet types and counts
df_wide = pd.DataFrame(
    data=[
        ["cat", 4, 5],
        ["dog", 7, 4],
        ["rabbit", 2, 3],
    ],
    columns=["pet", "male", "female"],
)

df_wide # view it
```

Without arguments, `melt()` collapses all columns into variable-value pairs.

```{python}
df_wide.melt() # melt all cols into variable-value pairs (pivot longer)
```

Specify identifier and value columns for a structured long-format result.

```{python}
melted_df = pd.melt(
    df_wide,
    id_vars=["pet"], # keep 'pet' as identifier column
    value_vars=["male", "female"], # value columns
    var_name="sex", # name for the new variable column
    value_name="count", # name for the new value column
)
```

## Pivoting Data

**Pivot: long → wide**  

Pivot the long data back to wide format:

```{python}
df_pivot = melted_df.pivot(
    index="pet", # identifier column becomes index
    columns="sex", # categories in variable column becomes new columns
    values="count", # value column becomes new values
).reset_index() # reset index to turn 'pet' back into a column

df_pivot.columns.name = None # remove the name of the columns index

df_pivot # view the pivoted DataFrame
```

Use `pivot_table()` instead of `pivot()` when there are duplicate index/column combinations, and specify an aggregation function like `aggfunc='mean'` to summarize duplicates.

For example, if we have duplicate pet entries:
```{python}
# create a sample DataFrame with duplicate pet entries
df_long = pd.DataFrame(
    data=[
        ["cat", "male", 4],
        ["cat", "female", 5],
        ["dog", "female", 7],
        ["dog", "male", 4],
        ["rabbit", "male", 2],
        ["rabbit", "female", 3],
    ],
    columns=["pet", "sex", "count"],
)
df_long # view the long-format DataFrame with duplicates
```

To pivot this long data back to wide format, we can use `pivot_table()` to handle duplicates:

```{python}
# create a pivot table that averages counts for duplicate entries
(df_long
.pivot_table(
    index="pet",
    columns="sex",
    values="count",
    aggfunc="mean", # average counts for duplicate entries
)
.rename_axis(None, axis=1) # remove the name of the columns headers
.rename_axis(None) # remove the name of the index
)
```

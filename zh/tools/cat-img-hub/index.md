# 猫咪图片收藏站

猫咪图片收藏站是一款浏览随机猫咪图片的网站，用户可以登录后收藏自己喜欢的猫咪图片，基于 **Next.js**、**TypeScript** 和 **MongoDB** 构建，使用 Cat API 和 Google OAuth 认证登录。

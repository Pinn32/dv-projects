# DV Projects - Personal Website

# Getting Started

## Local Preview

**1. run dv-dev conda env**

```zsh
conda activate dv-env
```

or

```zsh
source .zshrc
```

**2. quarto preview locally**

run 

```zsh
quarto preview
```

## Deploying to GitHub Pages

run 

```zsh
quarto publish gh-pages --no-render
```
